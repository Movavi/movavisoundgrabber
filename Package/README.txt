Пакет собирается из каталога Package/kexts, kext-бандлы туда нужно скопировать
вручную, т.к. на данный момент сборка всех необходимых версий в одном месте не
поддерживается. Т.е. Universal нужно собирать LLVM-GCC, а Signed собирается
CLang под Mac OS X 10.9.

В Signed лежит версия приготовленная для подписи, но не подписанная.
